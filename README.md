# Supplementary Materials: Quantum-Inspired Modeling of Oscillatory Task Substitution in Smart-Grid Automation

**Repository for the paper**  
**"Quantum-Inspired Modeling of Oscillatory Task Substitution in Smart-Grid Automation: Implications for Labor Reallocation in Renewable Energy Transitions"**  
by Agung Trisetyarso, Buyung Sofiarto Munir, and Kridanto Surendro

---

## Overview

This repository contains all open-source code, data, and scripts needed to fully reproduce the results and figures presented in the manuscript.  
It implements the **Quantum-Inspired Computational Intelligence Framework (QICIF)** extended to smart-grid automation, including:

- QuTiP-based Monte Carlo simulations of oscillatory labor-capital task substitution
- Resonant LC-circuit analogy and entangled quantum-state modeling
- Empirical validation against U.S. and European grid-operator labor data (2015–2024)
- Figures 1–3 as shown in the paper

The code is written in Python and designed for full reproducibility.
